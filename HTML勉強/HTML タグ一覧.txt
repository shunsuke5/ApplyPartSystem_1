HTML タグ一覧

buttonタグ
　→ ボタンを作る。

　　<button></button>

divタグ
　　→ 単体では意味を持たないが、クラスを指定することでdivで囲った部分を
　　　ブロックレベル要素としてグループ化することができる。

　　<div></div>

linkタグ
　　→ 外部ファイルを読み込む際にその情報とファイルを記述する。
　　　link relにはそのファイルとの関係性を記述し(例:stylesheetはcssファイルという意味)
　　　hrefにはそのファイルのパスを記述する。

　　<link rel="ファイルとの関係性" href="ファイルのパス">

formタグ
　　→ フォームを表す。

inputタグ
　　→ formタグ内で様々なフォームの入力欄を作成する。

labelタグ
　　→ フォーム入力欄のキャプション(見出し)を表す際に使用する。label要素で指定するキャプションと
　　　フォームの各入力欄を関連付けるには、「label要素のfor属性の値と、input要素、select要素、
　　　textarea要素などのフォーム部品のid名を同じものにする方法。2つ目の方法は、label要素自体の中に
　　　フォーム部品を配置すること。

<input type = radio>
　　→ ラジオボタンの入力欄を表す際に使用する。

<formtarget = "_self">
　　→ データ送信先の開き方を現在のウィンドウ、現在のフレームに開く。
　　　_brankなら新規ウィンドウに開く。
